spatial complexity masking (0.0-1.0)

0.0 disabled (default)
0.0-0.5 should be a sane range

larger values help against blockiness, if no deblocking filter is used for decoding

Tip: crop any black borders completly away as they will reduce the quality of the MBs there

normalize adaptive quantization (experimental)

When using adaptive quantization, the average per-MB quantizer may no longer match the requested frame-level quantizer. Using this option will attempt to adjust the per-MB quantizers to maintain the proper average.
