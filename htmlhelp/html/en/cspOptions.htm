Advanced color space options

Note : This feature is not in the svn nor in the patch section. This is a preview version to discuss how the feature and this document should be.
The advanced color space options page gives you control over the YCbCr <-> RGB conversion. The top half gives you control over the ITU-R BT.601/709, while the lower half is used for black and white cutoff, which control the contrast.

YCbCr specification

ITU-R BT.601/709 are ITU-R standard for component digital video. They define how to convert YCbCr <-> RGB.

  • ITU-R BT.601 : Almost all SDTV use this.
  • ITU-R BT.709 : Slightly different color compared to BT.601. HDTV use this. Please note that some HD clips prepared for computer may use ITU-R BT.601 color space.

Contrast

Almost all the video are encoded using Y (luma) range 16-235. YCbCr(16,128,128) is normally converted to RGB(0,0,0).
Some video clips have Y below 16 which represent blacker than black. If you want to take advantage of this gradation at the expense of contrast, use PC or custom mode.
