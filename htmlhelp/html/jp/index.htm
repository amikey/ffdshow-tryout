更新履歴

オリジナルのffdshow projectの最新バージョンからマルチスレッドパッチを取り除いた状態を基準としています。

不具合の修正

beta4以降の修正

  • VistaのWindows Media Centerでピクセルのアスペクト比が1:1以外のAVIファイルの再生に部分的に対応。Windows Media Centerにバグがあるため「再生中の出力形式の変更を許可する」と「互換フィルターにのみ接続する」の両方にチェックを入れる必要があります。
  • Raw videoでNV12/NV21を入力としてリサイズを使用するとクラッシュしていた。
  • 「ビットマップのオーバーレイ」がクラッシュしていた。
  • 再生用画像フィルタの「離散コサイン変換」が機能していなかった。
  • 字幕で「同時の字幕をオーバーラップする」が有効になっているとクラッシュすることがあった。
  • 字幕のレターボックスで再生中のビデオよりワイドなアスペクト比を指定すると上下が切れていた。
  • Windows Media Centerでサムネイル作成時にクラッシュしていた。

beta4以前の修正

  • FLACとTTAで映像との同期が乱れることがあった。
  • クローズドキャプションの文字化けを修正。
  • Windows Movie Makerでクラッシュしていたのを修正。
  • 字幕のファイル名に &が含まれているとトレイアイコン右クリック時のコンテキストメニューに正しく表示されなかった。
  • MakeAVIS.exeの"Store uncompressed audio"モードで16bit整数以外のフォーマットが正しく書き出せなかった。
  • 複数行のAviSynthスクリプトを「すべての設定を書き出す」やプリセットの「ファイルに保存」ボタンで保存出来なかった。
  • DTSのS/PDIFパススルーでノイズなどの問題が出ることがあった。
  • VFWエンコーダとアプリケーション（HyperCamなど）の相性問題を修正。
  • AviSynthのスクリプトでConvertToRGBを使用すると画像が反転していた。
  • オーディオフィルタの「チャンネルの入れ替え」を使用時に、入力ストリームのチャンネル数などが変化すると無音になっていた。
  • DVDのメニューがハイライトされないことがあった。
  • プリセットの自動読み込み機能「movie file name match (with wildcards)」が機能していなかった。
  • 入力色空間がNV12/NV21の時、正しく表示できなかった。
  • 字幕のレターボックスの後にCropを使用すると上端を切るように設定しても下端が大きく切れていた。

beta3以前の修正

  • Avisynthとインターレース解除で本来黒い縁が黒く塗りつぶされていなかった。
  • 漸進的ノイズ低減(Gradual denoise)を横幅が16の倍数以外のビデオで使用すると、クラッシュするか右端にグレーの帯が見えていた。
  • カーネルインターレース解除のメモリ・リークを修正
  • libavcodecのエンコーダが25fps以外のフレームレートを受け付けなかった。他のエンコーダでも微妙にフレームレートがずれるなどの不具合があった。
  • AviSynthのスクリプトでffdshowaudio()を使うとエラーが発生していた。
  • UNICODEの字幕に0x1aが含まれていると字幕が途中で切れてしまっていた。
  • AviSynthのフィルタがRBG色空間で出力すると画像が上下反転していた。
  • VFWデコーダのRaw video-YV12が有効になっている時にDirectShow経由でVFW codecが使用されると画像が上下反転していた。
  • Winampプラグイン安定性の改善。
  • 再生中にリサイズのダイアログで高さに（800の最初の一文字として）8を入力するとクラッシュしていた（メソッドによる、lanczosならタップが4以上のとき）。
  • リサイズ使用中にビデオのピクセルのアスペクト比率（SAR）のみが変わると無視されていた。
  • リサイズで内側の境界線（縁）が0より大きい数字に設定されている場合、「オリジナルのアスペクト比率を維持する」が機能していなかった。
  • AviSynthのスクリプトが保存出来なかった（Unicode版のみ）。

beta2以前の修正

  • DVDなどでアスペクト比が変更された後、画像の高さが半分になっていた（オーバーレイミキサとYUY2使用時）。
  • FFV1デコーダがシーク時、遅延時にクラッシュしていた。
  • FFV1エンコーダの「キーフレームの間隔」がデフォルトで250と遠すぎ、かつ独立して設定できなかった。新しいデフォルトは10。
  • DirectShowデコーダでNV12が唯一の出力色空間で、vfwデコーダのRaw videoが有効になっているとWindows Media Player11がサムネイル作成時にクラッシュしていた。
  • 字幕に白を指定しても灰色だった。
  • 現在使用中でデフォルトのプリセット設定を削除するとクラッシュしていた。
  • NV12で映像を反転すると画面が乱れるかクラッシュしていた。
  • Media Player Classicの内蔵matroskaスプリッタでアスペクト比が反映されなかった。
  • 再生中にオーディオのダイアログを開くとCPU使用率が異常に高く、ADPCMなどではフリーズすることがあった。
  • Vixがサムネイル作成時にクラッシュすることがあった。
  • DVobSub(VSfilter)との接続時にYV12、横幅が8の倍数以外を指定してクラッシュすることがあった（リサイズ使用時）。
  • リサイズ、アスペクト→縁取りで内側、100％を指定するとクラッシュしていた。
  • VMR9との再接続（リサイズ変更時など）に失敗することがあったのを修正。
  • PowerDVD7でDirectShowフィルタ接続の不具合のため画面が真っ暗になっていたのを修正（情報とデバッグ→以下ではffdshowを使用しないにPowerDVD.exeが含まれていたら削除して下さい）。
  • 字幕の行数が多くて指定位置からでは表示しきれない場合に切り捨てられていた。
  • イタリックの字幕で右端が切れていた。高速レンダリングがオンになっているとイタリックになるべきものがならなかったり、逆になったりしていた。
  • 一部のSVQ3で再生開始時にフリーズしていた。
  • matroska(*.mkv)の埋め込み字幕を切り替えようとするとクラッシュしていた。
  • matroska(*.mkv)の埋め込み字幕の日本語などが文字化けしていた。
  • wmv9のエンコーダ設定ダイアログで間違った設定が表示されていた。
  • ノイズフィルターの後にリサイズを行うと緑一色になっていた。
  • Windows Vistaでエクスプローラーがクラッシュしていた（AVIファイルがあるフォルダにアクセスした時）。
  • MPEG1/2で再生開始時およびシーク時にアーチファクトが出たりクラッシュしたりすることがあった(libmpeg2)。
  • oggとogmのmuxerを修正（flvとmpegのmuxerは削除）。
  • 高さが4の倍数でない動画を再生すると出力色空間によってクラッシュしたりアーチファクトが出たりすることがあった。
  • 幅1024ピクセルを超えるRGB→YUY2変換でクラッシュしていた。
  • YV12が有効になっていると奇数ラインの動画再生時に画像が乱れた。
  • HaaliのビデオレンダラとMPCの間にffdshowが入ったときにうまくシークできないことがあった。
  • Kernel deinterlaceの次の内部フィルターが１フレームおきにスキップされていた。次の内部フィルターがリサイズであった場合フレーム毎にビデオサイズが変化していた。

beta1までの修正

  • Windows Media Encoderでファイルの変換を行う時にffdshowがオーディオデコーダとして使用されクラッシュしていた。
  • libmpeg2でハードウエア・デインターレーシングが無効だった。
  • AACの再生が早すぎて聞き取れなかった、ビデオと同期しなかった。
  • トレイ、ダイアログとパスの設定がJapanese(UNICODE)になっていると文字化けしていた(ANSIビルド)。インストーラーがデフォルトでJapanese(UNICODE)を設定するので問題であった。
  • 奇数ラインの動画再生時にフリーズした。
  • ミキサー・ヘッドフォンの仮想空間化が有効になっているとモノラルの再生時にクラッシュした。
  • オンスクリーンディスプレーで全角文字を含むファイル名を表示させると途中で切れるなど日本語文字列処理に問題があった（主としてセキュリティ上の問題）。
  • XvidのN-VOPフレームに字幕などが重なると表示できなかった。ファイルの終わりに長いN-VOPフレームの連続があると早期終了してしまった。
  • 日本語ダイアログの右側が３０％程度切れていた（ダイアログの起動方法による）。
  • 「ビデオデコーダーの設定」-「出力」にある"DV"のチェックが外せなかった。
  • インストール時に更新が必要なファイルがロックされているとインストール出来なかった。
  • Didee's YlevelsG/S/Cが正しく実装されておらず暗部中心の補正にならなかった。
  • Uncompressedオーディオの入力が無効だった。
  • LPCM出力が出来なかった（オーディオデコーダの設定-出力）。
  • AC3出力が出来なかった（オーディオデコーダの設定-出力）。
  • DTSで大きなノイズが発生し再生できなかった。
  • 遠近法修正でキュービック補間を選択して幅768ピクセル以上の画像を処理させるとクラッシュしていた。
  • 5.1ch Vorbisの再生が出来なかった（libavcodec使用時）。
  • ffdshowがDVDデコーダの出力をraw videoで入力する場合、字幕のファイルに何か設定されていると字幕を無効にしても映画と関係のない字幕が表示されていた（mpeg2設定が無効などの場合）。
  • 字幕のフォント-高速レンダリングがオンになっていると日本語シフトJISが文字化けしていた。
  • 画像形式の字幕（IDX+SUB形式など）で字幕の高さが奇数ラインのところにくるとクラッシュしていた。
  • Convolver（オーディオフィルターの一つ）でインパルスファイルをロードしようとするとクラッシュしていた。
  • リサイズで水平方向にライン（アーチファクト）がみられることがあった。
  • Xシャープ化とリサイズを同時に使うと緑一色になっていた。
  • ステレオスコーピックの字幕でクラッシュしていた。
  • マルチプロセッサー環境でOSD-CPU使用率が正しく表示されなかった。
  • マルチプロセッサー環境でポストプロセッシング-クオリティの自動コントロールが機能しなかった。
  • ボリュームノーマライズが正常に機能していなかった。
  • DScaler: Video (TomsMoComp)を使用するとクラッシュしていた（ビルドに使用したコンパイラによる）。
  • オーバーレイ・コントロールで初期値が正しくロードされなかった。
  • oblivion, morrowind（いずれもゲーム）でクラッシュしていた。
  • Windows Vistaのダイアログでコーデックの選択枝が上のほうに表示されていた。
  • AviSynthのスクリプトにエラーがあるとOSD表示されるが、そのエラーメッセージがAviSynthを無効にしても消えなかった。

新機能

beta4以降の追加

  • Four CCのSP51-SP57を追加
  • SSA/ASSでも「アスペクト比を自動的に補正する」の設定を有効にした。vsfilterとMPCのVMR9 renderlessモードで動作が異なるために発生している互換性問題に対応したもの。

beta4以前の追加

  • Nellymoserオーディオデコーダーの追加。最新のFLV Splitterが必要です。
  • Seb's BTB&WTWレベルフィルタ：設定により最暗部(Y < 16)、最明部(Y > 235)の情報がある時だけそれを生かすコントラスト調整を行います。
  • H.264-PAFF インターレースフォーマットのサポート。
  • H.264デコーダのマルチスレッド化。ただしマルチスライスでエンコードされている場合にのみ有効。
  • オーバーレイコントロールにリセットボタンを追加。
  • ffavisynth.dllがAviSynth起動時に読み込まれるようにした。LoadCPlugin / Load_Stdcall_Plugin関数を使わずにffdshow()、ffdshowAudio()関数を使用できます。
  • YCbCr<->RGB変換にITU-R BT.601/709の選択、コントラスト（標準、フルレンジ、カスタム）のオプションを指定できるようにした。
  • 7.1chオーディオのサポート（フィルタ群）。
  • 切り出し（クロップ）フィルタに「自動切り出し」を追加。上下左右の黒い縁を自動で除去します。
  • プリセットの自動読み込みで「条件が一つでも一致した時（OR）」、「すべての条件が一致した時（AND）」を選択可能にした。
  • プリセットの自動読み込み「on FOURCC from previous filter match」を追加。Raw videoを処理する場合で元のFORCCが関係する場合に有効です。
  • Windows VistaのWindows Media Player 11とWindows Media CenterでRaw videの処理を有効にするオプションを追加した。
  • 「リサイズ、アスペクト」-「スクリーンの解像度にリサイズする」の追加。
  • 「リサイズ、アスペクト」-「ピクセルのアスペクト比率を内部的に処理する」で表示デバイスのアスペクト比率を設定できるようにした。
  • 「リサイズ、アスペクト」-「縁」で縁の分け方を指定できるようにした。字幕を使用する場合などに便利です。
  • 字幕の垂直方向の拡大・縮小をサポート。SSA/ASSの字幕もオプションにより拡大・縮小可能です。
  • 字幕のぼかしをサポート。背景に良く溶け込みます。
  • 字幕の折り返しを3モードサポート。「出来るだけ均等に分割」、「幅いっぱいになったら改行」、「出来るだけ均等に分割、下の行を広めに」。日本語禁則処理は今後の課題。
  • テキスト字幕とOSDのレンダリングを改善。文字、輪郭、陰影のそれぞれにサイズ、色、アルファを設定可能。RGB出力で字幕より下流にOSD、取り込み、オーバーレイ、リサイズ以外のフィルタがなければ字幕とOSDはRGB32でレンダリングされ画質がさらに改善します。
  • 多言語対応：言語ファイルから水平方向の拡大率を設定可能にした。水平方向を125％に引き伸ばしたければ、言語ファイルの[Font]セクションにhorizontalScale9x=125、horizontalScaleNT=125の2行を追加します。
  • OSDに「Active preset name」を追加。
  • 実験的ながらx64版のバイナリーをSourceforge.netのダウンロードページで公開。
  • x64版のRGB<->YV12変換を高速化。

beta3以前の追加

  • Xvidデコーダ(Xvidcore)のインターレース解除をサポート（インターレース関連フラグを設定）。
  • 互換アプリケーションリストに挙げられていないアプリケーションがffdshowをロードしようとした時、ダイアログを表示するようにした。
  • libavcodecのエンコーダ・エラーの原因を表示するようにした。
  • AviSynthフィルタのスクリプト内で現在のフレームだけでなく前後のフレームへの参照を可能にする「バッファー後方/前方」を追加。新たに追加された「3:2プルダウン」オプションとあわせてAviSynth用の高度な逆テレシネ変換やインターレース解除、時間軸(temporal)フィルタがffdshowできちんと作動するようになった。
  • アイコンを更新（ ）。旧アイコンも選択可能。
  • オーディオの圧縮フォーマット、ATRAC3（アトラック、Adaptive TRansform Acoustic Coding）に対応
  • Xvidエンコーダのマルチスレッド対応
  • Windows VistaのSPDIF問題に対応（オーディオデコーダーの設定->出力->接続先）。
  • Winampプラグインでマルチチャンネルオーディオを扱えるようになった。
  • リサイズの設定で「ピクセルのアスペクト比率（SAR）を内部的に処理する」機能を追加。
  • リサイズのGUIに「横幅を指定する」を追加（今まで高さに0を入力するGUIだったもの）。
  • オーディオデコーダーの出力形式としてマルチチャンネルストリームのみをAC3エンコードする機能を追加（直接SPDIF出力できないマルチチャンネル・オーディオを一旦AC3にエンコードしてAVアンプへ出力することを意図したもの）。

beta2以前の追加

  • 字幕の影の改善。3つのモードの新しい陰影が使用できます。
  • 互換性問題の管理，「以下でのみffdshowを使用する」の追加。
  • Raw videoのハードウエア・デインターレーシングのサポート（Raw videoで入力されたサンプルのインターレース関連情報を次のフィルタに伝えるようにした）。
  • OSDアイテム"IDCT(libavcodec)"の追加。オート時に何が選択されているか表示する。
  • 字幕ファイルの自動検索で拡張子の優先順位を指定できるようにした。
  • VC-1 Advanced Profileのサポートを追加（libavcodecは外部コーデックなしで動作可能）
  • 各種WMV、VC-1をMicrosoft製コーデックを使用して再生できるようにした(VC-1はコーデックまたはWMP11のインストールが必要）。
  • x264エンコーダーを更新して復活させた(tryoutsで一旦削除したもの)。
  • H.264 MBAFFのインターレース解除をサポート（インターレース関連フラグを設定）。
  • 外部プログラムに使用されるAPIの修正と新機能追加（プリセット、字幕、早送り・巻き戻しなど）
  • ポーランド語対応
  • キュー：VMR9に内蔵されているキューを利用できるようにした。

beta1までの追加

  • H.264 MBAFF（インターレース）のサポート
  • VP6他、いくつかのデコーダーを追加。
  • ダイアログの日本語化・多言語対応の改善。
  • インストーラーでスピーカー構成を設定できるようにした。OS（コントロールパネル）の設定をデフォルトとして使用します。
  • キュー：ビデオのタイムスタンプ待ちによるCPU時間の無駄を活用することにより高速化。
  • リサイズのマルチスレッド化：デュアルコアCPU等でリサイズを高速化。
  • Vorbis高音質モードを有効にした。
  • 画質と処理速度、同期との兼ね合いを設定できるようにした。
  • SVQ3のデコード時にスプリッタから十分な情報が得られなかった際エラーメッセージを表示するようにした。
